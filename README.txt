Założenia:

Plik CSV znajduje się w tej samej lokalizacji co skrypt
Plik CSV jest zatytułowany ship_ais.csv

Komendy:
Count Vessels - liczy ilość unikatowych numerów MMSI z nadajnikami typu A i B
Count MMSI - zwraca wszystkie unikatowe numery MMSI
quit - wyjście z programu
print data - drukuje kolejne rzędy i informacje z nimi powiazane
